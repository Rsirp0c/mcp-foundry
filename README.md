## Deployment

### GitHub Codespaces

- Open this repo in Codespaces and run:
  ```sh
  python __main__.py
  ```

### VS Code

- Open in VS Code and run:
  ```sh
  python __main__.py
  ```

### Docker

- Build and run:
  ```sh
  docker build -t mcp/foundry .
  docker run -it --rm mcp/foundry
  ```
